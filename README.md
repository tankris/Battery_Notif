# Battery_Notif
Battery Notification script for use with i3 WM

### Features

The features included in the currect script include:
* Low battery notification alert at 40%
* 80% and above charged notification alert
* 100% charged notification alert
* Doesn't show notification when steam process runs
* Injects auto-startup line into the i3 config (Tested only on Manjaro i3)

### Dependency

Please ensure that dunst notification daemon has been installed on the distro.

To check on arch or arch based distros, use ````pacman -Qs dunst````

Similarly, to install on an arch or an arch based distro, use ````sudo pacman -S dunst````

For more information on dunst, please refer to [Dunst - Arch wiki](wiki.archlinux.org/index.php/Dunst)

### Distro Compatability

Kindly note that the script currently has been tested only on Manjaro i3 Community Edition

Although the script isn't distrbo specific, in case the user has issues running the script on a different distro, kindly bring it up to the developer
